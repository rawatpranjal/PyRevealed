Theory
======

.. note::

   This section provides formal mathematical definitions for all methods in PyRevealed,
   based on *Revealed Preference Theory* by Chambers & Echenique (2016).

The theory documentation is organized into the following sections:

Overview
--------

**Budget-Based Analysis (Chapters 3-5):**

- :doc:`theory_foundations` - Notation, maintained assumptions, and axiom hierarchy
- :doc:`theory_consistency` - GARP, WARP, SARP and related consistency tests
- :doc:`theory_efficiency` - AEI, MPI, Houtman-Maks, and test power metrics

**Preference Structure (Chapter 4):**

- :doc:`theory_structure` - HARP, separability, utility recovery

**Menu-Based Analysis (Chapters 1-2):**

- :doc:`theory_abstract` - Abstract choice theory without prices

**Advanced Topics (Chapters 6-15):**

- :doc:`theory_advanced` - Integrability, welfare analysis, compensated demand
- :doc:`theory_extensions` - Spatial preferences, stochastic choice, limited attention, production
